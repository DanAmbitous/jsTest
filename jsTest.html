JS Explanation

JS is a programming langauge used along with HTML and CSS, it creates dynamic effects on websites, although it can also be used for creating native applications, games, mangaging backend, AI, as well as many other things. Although for some of them you would need JS libraries, frameworks, etc. such as for backend, but a lot can also be made with just JS. It was created by Brendan Eich in 1995, in just 10 days.

Setup Page

In order to use JS, it needs to be implented to the HTML page, there are 3 ways to do it. One is inline way which when you directly type in to the HTML element like this 

<p>onclick="alert('The paragraph has been clicked')"</p>

Another way is to do it internally, which is putting the script tags inside the HTML.

The final way is the most reccommend way, and that is externally which is linking the JS file to the HTML one.

Data Types

JS has different data types, data types are different types of code, that are used for ceratian things

Strings

Strings are things such as text, they are enclosed in single or double quotes, if other data types are enclosed in single or double quotes they become strings as well.

Numbers

Numbers, are well numbers, regardless if they are decimals or integers. They don't have any quotation marks, they are just written by themselves.

Booleans

Boolean has two values. Either true or false.

Null/Undefined

Null and undefined, mean it doesn't exist, undefined is typcially given by default if there isn't a value. Null is put by purpouse so something a variable would be empty

Variables

Variables are a way to store data for later use. To use variables there are 3 keywords

var

The var keyword is a way to create a variable, it has a global scope which means that anywhere in the code it can be called.

var aString = "Hello world";
const aNull = null;

let

let is another keyword to use a variable, but it has a limited scope, so it wouldn't interfer or work from any place in the code

let aNumber = 10;
const aNull;

It could also be equal to undefined

const

const stands for constant it's a the last variable, and it can only be called once, which means that its value can't be changed later in the code, unlike the other variables, it could help by running into less bugs.

const aBoolean = true;

Comments

Comments are way to write in the code without it interfing with the output. It's useful for explaining the code and for testing purposes as it can comment out codes.

There are two types of comments. One of them is called inline comment which only effects one line, and the other one is muliti line comment which can effect mulitple lines
